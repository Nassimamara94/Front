VALEUR

OBSERVEZ le programmevaleurs.js et tentez de prévoir les valeurs finales de chaque variable.

var a = 2;

a = a -1;

a++;

var b = 8;

b+=2;

var c = a + b * b;

d = a * b + b;

e = a * (b + b);

f = a * b / a;

g = b / a * a;

Vérifiez vos prévisons en créant ce programme puis en le débeguant pour observer les valeurs des variables.
