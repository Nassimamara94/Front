Fonction Imbriquée et Closure

Garder à l'esprit que l'on peut appeler une fonction de n'importe ou, même à l'intérieur d'une autre fonction. br>les fonctions imbriquées sont souvent utilisées comme un moyen de garder du code bien organisé. Une grande fonction complexe est plus facile à comprendre si divisée en plusieurs sous fonctions.



commentaire du code du script

Il faut s'assurer que les valeurs de la variable utilisée dans la fonction ont eu porté correctes. Ici on a une référence erreur var not defined. En effet bien que la valeur de du var avariable est définie dans la même portée que les appels de fonctions, elle n'est pas défini dans les définitions dans la déclaration des fonctions (le code réel qui est exécuté lorsque les fonctions sont appelées.) Pour que ça fonctionne il faut passer la valeur de la fonction en tant que paramètres.
