La boucle while

La boucle while sert à répeter une ou plusieurs lignes de code. En algo la boucle while s'appelle Tant Que.Elle définit une liste d'instruction tant que une condition est vraie.

Structure de la boucle while

                while(condition){
                    instruction 1;
                    instruction 2;
                }

While ne prend qu'un seul paramétre, qui est la condition à vérifier. Tant que la condition est vraie, la boucle est re-éxécuter. Il faut donc s'aranger un moment pour que la condition ne soit plus vraie, Sinon la boucle se répeterait indéfiniment. 

                 var x= 0; 
                 while(x < 20){
                      x+=5 ; /* ajoute 5 à la var x */ 
                    } 
                    console.log(x);
