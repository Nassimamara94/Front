Les boucles

Les instructions for et while qui sont 2 types de boucles les plus utilisées en js, servent à répéter plusiseurs ligne de code.

Le boucle For

For prends 3 paramètres encadrés dans des parenthèses, suivi entre accolades des instructions à exécuter autant de fois que la boucle l'indique. Les paramètres sont:

  • Initialisation : ici on déclare la variable qui servira de compteur
  • condition: tant que cette condition est vraie la boucle s'executera
  • Expression final: xpression qui est executer à chaque fin de boucle.
for(initialisation; condition; incrémentation){
instruction 1;
instruction 2;
}
