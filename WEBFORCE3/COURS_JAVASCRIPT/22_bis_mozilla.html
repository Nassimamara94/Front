La fonction definirMonUtilisateur cette fonction utilise la méthode() prompt qui affiche une boîte de dialogue, un peu comme une alert, sauf qu'elle permet à l'utiliateur de rentrer des données et de les enregistrées dans une variavble quand l'utilisateur click sur OK.
Dans cette exemple on demande à l'utilisateur de saisir son nom, ensuite nous appelons l'API localStorage qui permet de stocker des données dans le navigateur pour pouvoir les réutilisées ultérieurement.

Nous utilisons la fonction .setItem de cet API pour stocker la donnée qui nous intéresse dans un conteneur appelé nomUtilisateur.

La valeur stcockée ici est la valeur de la variable monNom qui contient le nom, saisi par l'utilisateur dans le prompt.

Au final on utilise la propriétée .textContent de monTitre pour lui affecter un nouveau contenu.

Le bloc de la condition if else utilise l'opérateur de la négation NOT "!" pour vérifier si le navigateur possède une donnée appelé nomUtilisateur ou pas.

Sinon la fonction definirMonUtilisateur est invoquée pour créer cette donnée.

Si elle exige "ce qui correspond au cas ou l'utilisateur est déjà venu sur la page, on récupère avec la méthode .getItem()".
Pour finir on définit le contenu de .textContent pour le titre avec une concaténation de chaine suivie du nom de l'utilisateur.

Enfin, on associe le gestionnaire d'événement click boutton. Quand l'utilisateur cliquera sur le boutton,ça déclenchera l'éxécution de la fonction definirMonUtilisateur();

Ce boutton permet donc à l'utilisateur de modifier son nom s'il le souhaite.
