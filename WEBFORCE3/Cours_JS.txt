le signe = est un opérateur d'assignation en javascript

Les variables : 

Une variable est un conteneur, une boite permettant de stocker des valeurs.

Pour commencer, il faut déclarer une variable avec le mot clé var en le faisant suivre du nom de la variable.

On déclare une variable en lui donnant un nom, on lui assigne une valeur avec l'opérateur d'assignation égale(=)

Exemple
a = 4; 
ou var a = 4;
(le mot clè var indique ici une variable globale)

let a = 4; c'est une déclaration de variable en ES6( à ne pas utiliser aujourd'hui dans du ES5)
