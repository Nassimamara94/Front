Selectionner en jQuery

Le coeur de jQuery se trouve dans la sélection d'éléments. Pour ensuite éxécuter du script dessus.

La commande $ comme selecteur accepte en argument tous les sélecteurs css

Nous renvoie ensuite une collection d'un ou plusieurs éléments DOM, enveloppés dans un objet jQuery

$('div') selectionne toutes les div

$('.titre') selectionne toutes les éléments avec la class titre

Cet enveloppe jQuery est celle qui donne acces à toutes les fonctionnalités de jQuery

  • rajouter des class
  • modifier du contenu HTML
  • Afficher ou cacher des éléments
  • Les annimer

Tout les éléments sélectionnés en js pure 'getElemntById', 'querySelector' ect, sont des éléments DOM et tous ceux sélectionnés avec $('') sont des éléments DOM dans une enveloppe jQuery

jQuery dispose de plusieurs méthodes pour parcourir l'arbre du document (DOM)

  • children() : parcourt les enfants direct
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie le parent
  • eq() renvoie parmi une liste d'éléments, celui qui se trouve à l'index fourni
  • children() sélectionne tous les enfants direct

les getters et setters

permet d'ajouter et retirer des class et savoir si on a déjà des class

  • addClass() : ajoute une class
  • removeClass() : retire une class
  • toggleClass : Interupteur ajoute/ retire une class
  • hasClass : tester l'existence d'une class
  • is() : tester si on a plusieurs class

Les événement

la plupart des interactions de l'utilisateur avec une page web peuvent être interceptés et une action lancé en retour.

On peut dire à une fonction de s'éxécuter, quand l'utilisateur click sur un bouton, ou bien quand il scroll dans la fenêtre, ou bien s'il appuie sur une touche du clavier.

  • mouse:
    1. au click sur l'élément
    2. mouse enter : lorsque la souris passe par dessus la zone de l'élément
    3. mouseleave: lorsque la souris sort de la zone de l'élément
  • keyboard:
    1. keyDown quand une touche de clavier est enfoncée
    2. keyUp lorsqu'une touche de clavier est relachée
  • window:
    1. scrool: le défilemnt de la fenêtre
    2. resize: le redimensionnement de la fenêtre
  • formulaire:
    1. change: pour les éléments <input>, <select>, et <textarea>, quand l'utilisateur change une valeur
    2. focus et blur: lorsque la souris entre et sort de l'input
  • submit: à l'envoie du formulaire
  • DOM:
    1. DOMContentLoaded : lorsque le DOM est complétement chargé et analyé sans attendre que les images et le css soient chargés
